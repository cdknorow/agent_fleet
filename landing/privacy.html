Privacy Policy

Coral by Subgentic.ai

At Subgentic.ai ("We," "Us," "Our"), we take your privacy seriously. This Privacy Policy explains what data we collect, why we collect it, and how we protect it when you use the Coral website (coralai.ai) and the Coral Desktop Application.

Your code, prompts, agent sessions, and workspace data never leave your machine. Coral processes everything locally.

1. What We Collect

1.1 Data We Collect Automatically

License Key Validation

When the Coral Desktop Application starts, it contacts our server to validate your license key. This request includes:

  • Your license key
  • A machine identifier (a hash derived from your hardware — not your MAC address or serial number)
  • The software version number
  • Your operating system and architecture (e.g., macOS arm64, Windows amd64)

This data is used solely to verify your license is valid and has not exceeded its activation limit. We retain validation logs for up to 90 days for abuse detection, then delete them.

Website Analytics

We use privacy-respecting analytics on coralai.ai to understand traffic patterns. We collect:

  • Page views and referral sources
  • Country-level location (no city or IP address)
  • Browser type and screen size
  • No cookies, no fingerprinting, no cross-site tracking

1.2 Data You Choose to Share (Opt-In Only)

Crash Reports (Opt-In)

If you enable crash reporting in the desktop application, we collect stack traces, error messages, software version, operating system, and a timestamp. Crash reports do NOT include your source code, agent output, prompts, file contents, workspace paths, or any data from your AI agent sessions.

Usage Analytics (Opt-In)

If you enable usage analytics, we collect which features you use, session duration, and number of agents run. Usage analytics do NOT include your source code, agent output, prompts, file contents, message board content, or any data from your AI agent sessions.

1.3 Data We Never Collect

We do not collect, transmit, or have access to:

  • Your source code or files
  • AI agent prompts, instructions, or output
  • Agent session content or terminal output
  • Message board messages between your agents
  • Git repository content, commit history, or branch names
  • File paths or directory structures on your machine
  • API keys or credentials for AI services (Claude, Gemini, etc.)
  • Screen recordings, screenshots, or keystrokes

Coral is a local-first application. All agent orchestration, communication, and monitoring happens entirely on your machine.

4. Data Security

We protect your data using HTTPS/TLS encryption for all data in transit, encrypted storage for license keys and customer records, access controls limiting data access to essential personnel only, and regular security reviews of our infrastructure.

No system is perfectly secure. If we become aware of a data breach affecting your personal information, we will notify you promptly as required by applicable law.

5. Your Rights

You may access your data, correct inaccurate information, delete your account, opt out of crash reports and usage analytics at any time via application settings, and export your data upon request.

If you are in the EEA (GDPR) or California (CCPA/CPRA), additional rights apply. Contact us for details.

6. Children's Privacy

Coral is not directed at children under 13. We do not knowingly collect personal information from children under 13.

7. Changes to This Policy

We may update this Privacy Policy from time to time. We will notify you of material changes by posting the updated policy on our website and updating the "Last Updated" date.
