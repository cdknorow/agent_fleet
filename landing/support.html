Common Issues

License Activation

  • After purchasing, you'll receive a license key via email from our payment provider (Lemon Squeezy).
  • Open the Coral app and enter your key on the activation screen.
  • Each license activates on 1 machine. To move your license to a new machine, deactivate it first in Settings.
  • If your key isn't working, double-check for extra spaces and make sure you're using the latest version of Coral.

Free Trial

  • The 14-day Pro Trial gives you full access with a limit of 2 Teams / 8 Agents.
  • No credit card required — get a trial key instantly from the trial signup page.
  • When your trial ends, upgrade to Pro to keep your teams and settings. Use code A3MDI0NQ for 50% off.

Installation

  • macOS: Download the DMG from coralai.ai, drag to Applications, and launch.
  • Linux: Download the tar.gz, extract, and run ./coral --host 127.0.0.1 --port 8420
  • Windows support is coming soon.

Agent Setup

  • Coral works with Claude (Anthropic), Codex (OpenAI), and Gemini (Google, in beta).
  • You'll need to authenticate each agent's CLI separately — Coral uses your existing API credentials.
  • See the documentation for step-by-step setup guides.
